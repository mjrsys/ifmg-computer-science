# 🎓 Repositório de Ciência da Computação - IFMG

Um repositório colaborativo criado por alunos de **Ciência da Computação do IFMG** para organizar e compartilhar materiais de estudo.

---

## 📚 Sobre

Este repositório tem como objetivo centralizar conteúdos acadêmicos de todos os períodos, incluindo:

* 📑 Slides das aulas
* 📝 Resumos e anotações
* 📋 Listas de exercícios e resoluções
* 📂 Trabalhos e projetos
* 🧠 Materiais de estudo

Nosso objetivo é criar uma **base de conhecimento organizada e duradoura** para ajudar alunos atuais e futuros.

---

## 🗂️ Estrutura do Repositório

O conteúdo é organizado por período e disciplina:

```
01-primeiro-periodo/
02-segundo-periodo/
03-terceiro-periodo/
...
```

Cada período contém as matérias, e cada matéria possui:

```id="q8m2pw"
slides/
notas/
exercicios/
trabalhos/
provas/
projetos/
```

---

## 📦 Outras Seções

* `shared-resources/` → Materiais gerais (programação, matemática, inglês, carreira)
* `livros/` → Livros e materiais (de preferência links)
* `extras/` → Roadmaps, dicas de estágio e conteúdos extras

---

## 🚀 Como Usar

1. Acesse seu período atual
